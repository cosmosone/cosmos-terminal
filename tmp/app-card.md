# Cosmos Terminal — App Card

**Platform badge:** Windows

## Short Description

A lightweight, native terminal for Windows built for vibe coding — project workspaces, split panes, an embedded browser, and Git all in one window. No IDE required.

## Selling Points

- **Everything in one window, nothing wasted** — Terminal, browser, file editor, and Git sidebar live together as tabs inside project workspaces. Switch between your dev server, AI agent, localhost preview, and source control without Alt-Tabbing or launching an IDE. Each project remembers its layout across restarts.

- **AI agents at your fingertips** — Launch Claude, Codex, Gemini, or Cline directly from the new-session dropdown. Agent sessions run as regular tabs alongside your terminals, so you can vibe code with AI while watching your build and previewing the result — all in the same project workspace.

- **Windows-native and lightweight** — Built with Rust and Tauri v2, not Electron. ConPTY for real shell support, WebView2 for the embedded browser, WebGL rendering with ClearType — everything uses what Windows already has. Small install, low memory, fast startup.
